---
title: 加载器
description: 用于页面和区块的加载中状态 - 页面局部处于等待异步数据或正在渲染过程时，合适的加载动效会有效缓解用户的焦虑。
---

## 导入

```js
import { Spinner } from "@resolid/react-ui";
```

## 用法

:::UsageBlock{file=spinner/spinner.tsx ignores=label}

import { Spinner } from "@resolid/react-ui";

export default function Usage(props) {
  return <Spinner {...props} />;
}

:::

## 举例

### 颜色

通过传递 `color` 属性来更改加载器的颜色。加载器支持 `primary`、`secondary`、`success`、`warning`、`danger` 和 `neutral` 这些颜色。

```jsx demo
import { Spinner } from "@resolid/react-ui";

export default function App() {
  return (
    <div className={"flex flex-row items-center gap-3"}>
      <Spinner color={"primary"} />
      <Spinner color={"secondary"} />
      <Spinner color={"success"} />
      <Spinner color={"warning"} />
      <Spinner color={"danger"} />
      <Spinner color={"neutral"} />
    </div>
  );
}
```

### 大小

通过传递 `size` 属性来更改加载器的大小。加载器支持 `xs`、`sm`、`md`、`lg` 和 `xl` 这些大小。

```jsx demo
import { Spinner } from "@resolid/react-ui";

export default function App() {
  return (
    <div className={"flex flex-row items-center gap-3"}>
      <Spinner size={"xs"} />
      <Spinner size={"sm"} />
      <Spinner size={"md"} />
      <Spinner size={"lg"} />
      <Spinner size={"xl"} />
    </div>
  );
}
```
