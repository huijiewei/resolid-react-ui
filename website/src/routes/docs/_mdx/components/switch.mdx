---
title: 开关
description: 互斥性的操作控件，用户可打开或关闭某个功能。
---

## 演示

```jsx demo online=Switch componentProps=switch/switch.tsx|color,size,disabled,readOnly,invalid
import { Switch } from "@resolid/react-ui";

export default function App() {
  return <Switch>开关</Switch>;
}
```

## 用法

```js
import { Switch } from "@resolid/react-ui";
```

## 特点

- 遵循 [WAI ARIA Switch](https://www.w3.org/WAI/ARIA/apg/patterns/switch/) 设计模式。
- 由本机 HTML `<input>` 元素构建，该元素在视觉上隐藏以允许自定义样式。
- 标记对辅助技术的支持。
- 可以是受控的，也可以是不受控的。

## 举例

### 默认值

使用 `defaultChecked` 属性来设置开关的默认状态。

```jsx demo
import { Switch } from "@resolid/react-ui";

export default function App() {
  return <Switch defaultChecked>开关</Switch>;
}
```

### 可控值

使用 `checked` 和 `onChange` 属性来控制开关的状态。

```jsx demo
import { Switch } from "@resolid/react-ui";
import { useState } from "react";

export default function App() {
  const [checked, setChecked] = useState(false);

  return (
    <div className={"flex flex-col gap-2"}>
      <p>飞行模式: {checked ? <span className="text-fg-success">已激活</span> : "未激活"}.</p>
      <Switch checked={checked} onChange={setChecked}>
        飞行模式
      </Switch>
    </div>
  );
}
```
